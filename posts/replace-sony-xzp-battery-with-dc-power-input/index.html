Replace SONY XZP battery with DC power input

Aug 10, 2023

Micro-USB port

  • Pin 1: VCC (Red)
  • Pin 2: D- (White)
  • Pin 3: D+ (Green)
  • Pin 4: ID (Optional?)
  • Pin 5: GND (Black)

SONY XZP Motherboard Battery Port

  • EB+ (Top) = VCC
  • ID
  • EB- (Bottom) = GND

LM2596S DC-DC Step-Down Voltage Power Supply Module Board

The input voltage needs to be at least 1.5V higher than the output voltage. For the first use, the potentiometer needs to be rotated counterclockwise for at least 20 turns. Counterclockwise rotation is for step-down voltage, while clockwise rotation is for boosting voltage.

Adjust the potentiometer along with the output voltage to 4.2 volts.

Final Step

Use a soldering iron to solder the above modules together.

Press the power button for 3 seconds to power on, while keeping an eye on the output power voltage of the LM2596S; it won’t work if the voltage goes below 3.8 volts.
