以下内容恢复自 Wordpress 时期的数据库备份，内容已经严重过期，仅留作纪念。

我打算把原先的 Rss 读取程序增强一下，变成可以自定义读取条数的。

于是就花了 1 个小时去处理了一下，因为考虑特殊情况，所以我给单数和复数条目定义了不同的 Class，同时请自行加上 UL 标签。

一. Get_More_RSS：

1. GMR 程序下载：

程序名 - Get_More_RSS （RSS文章标题链接读取器）

版本 - v1.07 修正增强版

$rssurl_save_time = 1296000; // RSS地址缓存时间，这个设置长一点儿没啥关系
$rsscont_save_time = 7200; // RSS内容缓存时间，这个设置看情况吧，请根据服务器负载自行调整
$rsscont_read_count = 5; // RSS读取出来的条数，按需求填写即可

GMR v107 程序代码下载传送门 - http://code.google.com/p/evlosbox/downloads/detail?name=get_more_rss.txt

GMR v107 GBK 版下载传送门 - http://code.google.com/p/evlosbox/downloads/detail?name=get_more_rss.txt

2. GMR 使用方法：

<script type="text/javascript" src="do.php?url=evlos.org"></script>

像上面这个样子的调用方法，很简单的，然后即可生成像文章顶部一样格式的 RSS 条目。
