如何给站点添加 Icon

Oct 7, 2009

以下内容恢复自 Wordpress 时期的数据库备份，内容已经严重过期，仅留作纪念。

网站的 Icon 是除 Logo 外的另一个图形标志，它不仅是网站身份的象征，还能在浏览器收藏夹中凸显出来。本站现在还是暂时使用 Inove 自带的 Icon，新的 Icon 正在制作中。

<img src="http://www.evlos.org/ev-box/myimg/uploads/Capture273.JPG" />

步骤

  • 只要编辑模板中的 Header.php
  • 找到标签
<title>

当然非 Wordpress 站点可能在 index 文件，实际上是 title 标签在哪就插哪，在其下方新建一行插入以下代码，显示页面的时候即会自动使用网站根目录的 favicon.ico 文件

<link rel="Shortcut Icon" href="/favicon.ico" type="image/x-icon" />

当然你也可以把以下函数加在 href 中，可用来指向当前模板根目录

<?php bloginfo(\'template_directory\');?>
