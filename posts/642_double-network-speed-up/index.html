双网卡提升下载速度

Oct 28, 2009

以下内容恢复自 Wordpress 时期的数据库备份，内容已经严重过期，仅留作纪念。

Windows7的网卡槽点

话说在 Windows 7 系统里，至今依然没有原生提供双网卡叠加加速上网的功能。因为通常一个 IP 只能对应一个客户端，所以像玩游戏时依然只能乖乖使用一张网卡。个人觉得系统如果能提供一个“进程绑定网卡”的小功能，肯定会爽翻天的说！

P2P下载加速实测

不过对于迅雷或电驴这样的 BT/P2P 客户端，我们倒是可以手动搞定带宽叠加加速。在我这里实测，自家的 4M ADSL 加上蹭来隔壁家的 2M ADSL，双管齐下直接可以加速到 550Kb/s 呢！

双网卡操作秘籍

想要实现这种网速叠加的效果，只需简单的几个步骤。

  • 开机后先暂时禁用掉其中一块网卡。
  • 接着启动迅雷并建立任务开始下载。
  • 等开始下载以后，再重新启用第二块网卡就可以让流量顺利走双线了。

关于微博的碎碎念

最后再附带一个小吐槽。虽然新浪微博经过长期的打磨终于结束了内测阶段。但是对于开发者而言，没有开放 API 的微博真的还有存在的意义吗？
