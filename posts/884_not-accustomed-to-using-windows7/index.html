不怎么习惯用 Windows7

Nov 9, 2009

以下内容恢复自 Wordpress 时期的数据库备份，内容已经严重过期，仅留作纪念。

╮(╯▽╰)╭ 真是怪咧，我就是死活不习惯用 Windows 7。各种纠结啊，难道我骨子里其实是个死忠“XP控”？神啊，赶紧来救救我吧！

Windows 7

界面确实炫酷得很，新版任务栏也很带感，很多细节功能肉眼可见地更完善了。但我个人实在是用不惯 Win7 的文件管理器，这点上还是更偏爱老当益壮的 XP。而且现在好多软件都没跟上 Win7 的步伐，兼容性显得有些尴尬。最让我心疼的是我的小本本，饱受新系统折磨，可怜的它居然因为显卡芯片过热直接强行关机了！

Ubuntu

这系统绝对是高效又自由，但咱毕竟还是个要玩玩游戏的人呐。我只想安安稳稳只装一个系统，纠结死我了，在上面迅雷居然跑不起来。网上还说什么用 Wine 加上国际版迅雷，在网络负载高的地方根本下不动东西嘛，实在太折腾了。

Mac OS

似乎还是卡在硬件驱动的问题上，目前只能继续观望中。

Windows XP

虽说很多功能现在看确实老旧了，但我个人感觉它的操作效率反而是最高的。只要你别手欠去装一堆杂七杂八的浏览器工具栏，别搞得开机启动项满天飞，它基本能非常顺滑地跑上好长一段时间，而且系统资源消耗也小。用 XP 的时候，我能真真切切地感觉到自己是完全掌控这个系统的，里面的每一个进程、每一个服务我都熟门熟路。不过话说回来，人总得赶赶潮流，XP 迟早是要被淘汰的，纠结半天，咱要不还是先把 Win7 凑合用着好啦。

唉，继续纠结中，到底装个什么系统才好哩？
